# Chatbot - Python - Flask - HTML - CSS - JavaScript
- Python

## Introduction

- Chatbot được xây dựng bằng Flask Framework

## Installation

```
# install requirements
pip install Flask
pip install CORS
pip install unidecode
pip install flask_cors
```

# RUN

```
$ python app.py
$ run index.html với Live Server hoặc (Go Live)
```
